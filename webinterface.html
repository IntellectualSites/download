What is the Schematic Web Interface?
The schematic web interface is a place to upload or download, manage and maintain schematics for your network.

  • Downloads/Uploads are entirely unlisted/anonymous
  • You can run it on a different machine to your server
  • You can connect multiple servers to the same web interface
  • You can upload files from your computer
  • You can upload files by using FAWE's build in command //download
  • Supports multiple formats (MCA/Schematic/Schem)
  • You can delete a file after you download it
  • You can customize the entire interface.
  • And lots of more features! Check the source
